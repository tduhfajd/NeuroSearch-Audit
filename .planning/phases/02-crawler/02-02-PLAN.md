# Phase 2, Plan 2: JS Rendering, robots/sitemap, and PageSpeed Providers

## Objective

Добавить JS-рендеринг через Playwright по эвристике, проверки `robots.txt`/`sitemap.xml`, и абстракцию PageSpeed-провайдеров (API primary + Lighthouse fallback).

## must_haves

1. JS-страницы корректно обрабатываются через Playwright, когда HTTP-парсинг недостаточен.
2. Наличие/доступность `robots.txt` и `sitemap.xml` фиксируются в метаданных аудита.
3. PageSpeed score собирается для топ-10 страниц по inlinks_count.
