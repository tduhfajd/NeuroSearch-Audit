# Phase 2, Plan 1: Crawler Core and HTML Extraction

## Objective

Реализовать базовый краулер по домену с гибридным источником URL (sitemap + homepage), ограничениями обхода и извлечением всех HTML-полей, необходимых для сохранения в `pages`.

## must_haves

1. Запуск краулинга обходит сайт до лимита страниц, начиная с sitemap или главной страницы.
2. Для каждой страницы извлекаются: status_code, title, h1, meta_description, canonical, robots_meta, ссылки (внутр./внешн.), json_ld, word_count, inlinks_count.
