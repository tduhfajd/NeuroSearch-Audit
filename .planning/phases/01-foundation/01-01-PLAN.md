# Phase 1, Plan 1: Foundation Bootstrap and API Skeleton

## Objective

Создать минимальный рабочий каркас проекта: базовые файлы окружения/зависимостей, FastAPI entrypoint, `/health`, `/health/db`, и skeleton-роутер `/audits` с валидацией схем.

## must_haves

1. `uvicorn backend.main:app` запускается без ошибок на порту 8000.
2. `GET /health` возвращает `{"status": "ok"}`.
3. Структура директорий и базовых файлов соответствует границам Phase 1.
