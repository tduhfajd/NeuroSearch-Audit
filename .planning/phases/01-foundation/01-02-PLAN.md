# Phase 1, Plan 2: Database Models and Alembic Baseline

## Objective

Сформировать базовую схему данных Foundation: модели SQLAlchemy (`audits`, `pages`, `issues`, `reports`), DB session, и первичную Alembic migration с созданием всех таблиц.

## must_haves

1. PostgreSQL запущен локально, все таблицы создаются через Alembic migration.
2. Структуры данных покрывают Foundation-часть требований (CRL-01 partial, SCR-01 data backbone).
