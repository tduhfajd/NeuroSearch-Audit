# Definition of Done: когда задача/фаза/милестоун "готовы"

Если пункт не выполнен — работа ещё не закончена.

---

## DoD для задачи (task)

- [ ] Задача реализована согласно `<action>` из PLAN.md.
- [ ] `<verify>` проходит успешно.
- [ ] Линтер/типизация чистые.
- [ ] Не выходит за scope задачи (нет "заодно").
- [ ] Атомарный коммит: `<type>(<phase>-<plan>): <описание>`.
- [ ] Нет секретов в коде/логах/коммите.

## DoD для плана (plan)

- [ ] Все задачи плана выполнены и закоммичены.
- [ ] SUMMARY.md создан с результатами, отклонениями и рисками.
- [ ] Полный набор тестов проходит.
- [ ] Нет стабов: TODO, FIXME, NotImplementedError, pass в бизнес-логике.

## DoD для фазы (phase)

- [ ] Все планы фазы выполнены (все SUMMARY.md созданы).
- [ ] Goal-backward верификация: все Success Criteria подтверждены evidence.
- [ ] VERIFICATION.md создан и показывает PASS.
- [ ] UAT пройден (все тесты pass или issues зафиксированы и исправлены).
- [ ] STATE.md обновлён (позиция, прогресс, решения).
- [ ] ROADMAP.md обновлён (фаза отмечена Complete).
- [ ] CHANGELOG.md обновлён.

## DoD для фазы с "опасными" изменениями (данные/auth/security/infra)

Дополнительно к DoD фазы:
- [ ] Тесты на негативные сценарии (ошибки, запреты, edge cases).
- [ ] Нет ослабления доступа/авторизации "для теста".
- [ ] Нет секретов в коде/логах.
- [ ] Rollback план описан (как откатиться при сбое).
- [ ] Валидация входных данных на границе системы.

## DoD для милестоуна (milestone)

- [ ] Все фазы милестоуна прошли DoD фазы.
- [ ] Аудит: все v1 requirements покрыты и verified.
- [ ] Нет стабов во всей кодовой базе.
- [ ] Полный тест-suite зелёный.
- [ ] MILESTONES.md обновлён с описанием, статистикой, git range.
- [ ] Git tag создан.
- [ ] Завершённые фазы архивированы в ROADMAP.md (collapsed).

---

## Чек-лист быстрой проверки (Quick Check)

Перед тем как считать что-то "готовым", пройди:

1. **Работает?** — Автотесты зелёные, UAT пройден.
2. **Чисто?** — Линтер, типы, нет стабов.
3. **Безопасно?** — Нет секретов, валидация на месте.
4. **Задокументировано?** — SUMMARY, STATE, CHANGELOG обновлены.
5. **Откатываемо?** — Атомарные коммиты, понятная история.

---

## Когда НЕ нужен полный DoD

### Quick Mode задачи
Минимальный DoD:
- [ ] Реализовано.
- [ ] Тесты зелёные (если применимо).
- [ ] Атомарный коммит.
- [ ] CHANGELOG обновлён.

### Прототипирование
Минимальный DoD:
- [ ] Работает (smoke test).
- [ ] Коммит с пометкой `chore: prototype`.
- [ ] TODO в коде допустимы.
