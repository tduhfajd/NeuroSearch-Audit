# Code of Conduct

## Наши принципы

Мы хотим поддерживать дружелюбное, безопасное и профессиональное пространство для всех участников.

Ожидаемое поведение:

- уважительное общение;
- конструктивная обратная связь;
- открытость к разным точкам зрения;
- фокус на решении проблем, а не на личности.

Недопустимое поведение:

- оскорбления, травля, дискриминация;
- угрозы и преследование;
- публикация чужих персональных данных без согласия;
- агрессивные или унижающие комментарии.

## Область действия

Этот кодекс действует в рамках репозитория, issues, pull requests и других каналов коммуникации, связанных с проектом.

## Реакция на нарушения

В зависимости от тяжести нарушения могут применяться:

- предупреждение;
- временное ограничение участия;
- удаление контента;
- постоянный бан из каналов проекта.
